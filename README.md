## 사용방법

1. 실행 후 왼쪽 중앙의 버튼을 눌러 패널을 엽니다.
2. '불러오기' 누릅니다.
3. 유니팩을 선택합니다. *유니팩 단락 참고*
4. `자동재생`을 누르거나 `스페이스 바`를 눌러 유니팩을 재생합니다.
5. 패드의 위치를 외웠다면, 마우스로 직접 패드를 눌러 즐겁게 연주 합니다.

> 참고!
> 런치패드가 있다면 런치패드로 연주할 수 있습니다.
> 별도의 설정이 필요 없습니다. 컴퓨터에 연결해 바로 연주하세요!
